# 시계
`date` 보다 이쁜 시계에요 😃

## 설치방법
`.zshrc`에 `/usr/bin/python3 (클론된 디렉터리 주소)/exec.py`를 추가시켜 줘요 😎

## 사용방법

### 형식
#### `watch [FLAGS/<blank>]`

### 플레그

#### `watch -h` or `watch --help`
`watch`의 도움말을 출력해요

### `watch -V` or `watch --version`
`watch`의 버전을 확인해요
